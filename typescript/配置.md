```jsonc
{
  "extends": "@workspace/typescript-config/react-library.json",
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@workspace/ui/*": ["./src/*"]
    }
  },
  "include": ["."],
  "exclude": ["node_modules", "dist"]
}

```
逐行解释：

**`extends`** — 继承公共 TS 配置，不用每个包重复写一堆配置项。

**`compilerOptions`**
- `baseUrl: "."` — 设置路径解析的根目录为当前目录
- `paths` — 路径别名，把 `@workspace/ui/*` 映射到 `./src/*`，这样导入时可以写：
  ```ts
  import { Button } from '@workspace/ui/button'
  // 实际解析为 ./src/button
  ```

**`include: ["."]`** — 编译当前目录下所有 TS 文件。

**`exclude`** — 排除 `node_modules` 和 `dist`，不参与编译。
